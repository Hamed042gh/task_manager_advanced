## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Hamed042gh/task_manager_advanced.git
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Run the development server:
    ```bash
    npm run dev
    ```
